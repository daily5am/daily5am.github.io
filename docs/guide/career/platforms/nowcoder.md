# 牛客网

## 平台简介

牛客网成立于2014年，是中国领先的IT行业招聘平台，专注于IT、互联网技术岗位的招聘服务。作为技术氛围浓厚的专业平台，牛客网以技术岗位多、算法题练习为特色，为技术人员提供专业的求职和技能提升服务。

## 平台特色

### 技术专业
- **专注IT行业**: 专注于IT和互联网技术岗位
- **技术岗位多**: 汇聚大量技术岗位
- **技术氛围浓**: 技术氛围浓厚
- **技术交流**: 提供技术交流平台

### 技能提升
- **算法题练习**: 提供大量算法题练习
- **技术竞赛**: 举办各类技术竞赛
- **技术博客**: 提供技术博客和分享
- **技能认证**: 提供技术技能认证
