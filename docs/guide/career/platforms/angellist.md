# AngelList

## 平台简介

AngelList成立于2010年，是全球领先的初创公司和创业招聘平台，专注于初创公司和创业生态的招聘服务。作为创业背景的招聘平台，AngelList以创业公司多、股权激励机会为特色，为有创业精神的求职者提供独特的职业发展机会。

## 平台特色

### 创业生态
- **专注初创公司**: 专注于初创公司和创业公司
- **创业公司多**: 汇聚大量创业公司
- **股权激励**: 提供股权激励机会
- **创业文化**: 体现创业文化和精神

### 创新导向
- **创新文化**: 体现创新文化和理念
- **快速成长**: 支持快速成长和发展
- **风险承担**: 适合愿意承担风险的人员
- **机会多样**: 提供多样化的职业机会
