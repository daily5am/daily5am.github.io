# 深圳市金融科技协会

## 协会简介

深圳市金融科技协会是全国最活跃金融CIO社群，涵盖深交所、中国平安等知名机构。协会致力于推动金融科技的发展，为会员企业提供技术交流、政策支持、人才培养等服务。

## 核心优势

- **社群活跃**: 全国最活跃金融CIO社群
- **机构权威**: 涵盖深交所、中国平安等知名机构
- **技术领先**: 汇聚金融科技领域顶尖企业
- **政策支持**: 获得政府大力支持
