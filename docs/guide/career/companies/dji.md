# 深圳市大疆创新科技有限公司

## 企业概述

**深圳市大疆创新科技有限公司**（DJI Technology Co., Ltd.）是全球领先的无人机制造商和无人机技术公司，总部位于中国广东省深圳市南山区。

- **成立时间**：2006年
- **创始人**：汪滔
- **总部地址**：广东省深圳市南山区科技园南区大疆创新大厦
- **企业性质**：非上市公司
- **注册资本**：1000万元人民币
