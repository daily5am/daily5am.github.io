# 字节跳动有限公司

## 企业概述

**字节跳动有限公司**（ByteDance Ltd.）是全球领先的互联网科技公司，总部位于中国北京市海淀区中关村。

- **成立时间**：2012年
- **创始人**：张一鸣
- **总部地址**：北京市海淀区中关村大街1号海龙大厦
- **企业性质**：非上市公司（计划IPO）
- **注册资本**：1000万元人民币
