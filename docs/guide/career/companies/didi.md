# 滴滴出行科技有限公司

## 企业概述

**滴滴出行科技有限公司**（Didi Chuxing Technology Co., Ltd.）是全球领先的一站式出行平台，总部位于中国北京市海淀区。

- **成立时间**：2012年
- **创始人**：程维
- **总部地址**：北京市海淀区中关村软件园二期
- **上市地点**：纽约证券交易所（DIDI）
- **企业性质**：上市公司
