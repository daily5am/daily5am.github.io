# 阿里巴巴集团控股有限公司

## 企业概述

**阿里巴巴集团控股有限公司**（Alibaba Group Holding Limited）是中国领先的电子商务和云计算公司，总部位于中国浙江省杭州市余杭区文一西路969号。

- **成立时间**：1999年
- **创始人**：马云
- **总部地址**：浙江省杭州市余杭区文一西路969号阿里巴巴西溪园区
- **上市地点**：纽约证券交易所（NYSE: BABA）、香港联交所（09988）
- **企业性质**：上市公司
