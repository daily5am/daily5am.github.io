# 华为技术有限公司

## 企业概述

**华为技术有限公司**（Huawei Technologies Co., Ltd.）是全球领先的信息与通信技术（ICT）解决方案供应商，总部位于中国广东省深圳市龙岗区坂田华为基地。

- **成立时间**：1987年
- **创始人**：任正非
- **总部地址**：广东省深圳市龙岗区坂田华为基地
- **注册资本**：403.08亿元人民币
- **企业性质**：民营企业

## 投资机构

华为技术有限公司为**非上市公司**，主要股东包括：

- **华为投资控股有限公司**：100%控股
- **华为工会委员会**：员工持股平台
- **任正非**：个人持股约1.01%

华为采用员工持股制度，约15万名员工通过工会委员会持有公司股份。
